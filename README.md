### Golang + Vue3 加密币模拟交易所

线上地址：https://eipistar.us.kg/

![eipistar.png](resource%2Fimages%2Feipistar.png)

#### 本地运行步骤:

1、安装依赖：npm install

2、运行项目：npm run prod

3、处理cookie跨域：打开控制台，修改cookie的domain。将www.eipistar.com修改为localhost，刷新当前页面即可。
